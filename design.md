# Portfolio Website Design Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a2332) - Professional, trustworthy, technical
- **Secondary**: Warm Copper (#b87333) - Sophistication, leadership, warmth  
- **Accent**: Soft Sage (#8fbc8f) - Growth, balance, innovation
- **Neutral**: Light Gray (#f8f9fa) - Clean, modern, readable

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings, conveys expertise and authority
- **Body Font**: "Inter" - Clean, modern sans-serif for excellent readability
- **Code Font**: "JetBrains Mono" - For technical content and code snippets

### Visual Language
- **Minimalist Professional**: Clean layouts with purposeful white space
- **Technical Sophistication**: Subtle tech-inspired patterns and geometric elements
- **Leadership Focus**: Strong typography hierarchy and confident color usage
- **Data-Driven**: Visual representations of skills and achievements

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth animations for skill wheel and timeline interactions
- **ECharts.js**: Professional data visualizations for skills radar chart
- **Typed.js**: Typewriter effect for hero section introduction
- **Splitting.js**: Text animation effects for section headings
- **Splide.js**: Image carousel for project screenshots
- **p5.js**: Subtle particle background effect for hero section
- **Pixi.js**: Interactive skill wheel with hover effects

### Animation & Styling
- **Hero Background**: Subtle particle system with floating geometric shapes in navy and copper
- **Text Effects**: Typewriter animation for name and title, letter-by-letter reveal for section headings
- **Skill Wheel**: Interactive radar chart with smooth transitions and hover details
- **Timeline**: Horizontal scrolling timeline with animated project nodes
- **Scroll Motion**: Gentle fade-in animations (150-300ms) with 16-24px vertical translation
- **Hover Effects**: 3D tilt on cards, shadow expansion, subtle glow on interactive elements

### Header Effect
- **Background**: Animated gradient mesh using CSS and JavaScript
- **Color Flow**: Subtle transition between navy, copper, and sage tones
- **Geometric Elements**: Floating triangles and hexagons representing data structures and cloud architecture

## Layout Principles
- **Grid-Based**: Consistent 12-column grid system
- **Responsive**: Mobile-first design with breakpoints at 768px, 1024px, 1440px
- **Vertical Rhythm**: Consistent spacing using 8px base unit
- **Content Hierarchy**: Clear visual hierarchy with size, color, and spacing
- **Interactive Feedback**: Clear visual feedback for all interactive elements
